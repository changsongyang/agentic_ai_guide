# 第二章 规划与推理引擎

智能体的核心能力在于推理与规划。本章讲解如何让智能体进行复杂推理、分解任务、反思改进，并掌握智能体专属的提示词工程。从思维链(CoT) → 任务分解(ToT/GoT) → ReAct闭环 → 自我反思(Reflexion)的进阶路径，最终通过系统化提示词工程实现行为规范化。

## 学习目标

完成本章后，你将能够：

1. **运用** 思维链(CoT)与任务分解算法解决复杂问题
2. **实现** ReAct框架与Reflexion自进化机制
3. **编写** 高效的智能体系统提示词，规范行为与约束
